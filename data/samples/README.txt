- acme_annual_report_2024.txt      → plain text with hyphen-newline for normaliser tests
- beta_q3_2024.md                  → markdown trading update
- finance_snippets.jsonl           → line-delimited { "text": ... } records
- financials_quarterly.csv         → quarterly time series (revenue, margin, EPS)
- esg_emissions.csv                → annual Scope 1/2/3 totals
- kpis.xlsx                        → two sheets: timeseries & ratios
- press_release.json               → small JSON with finance highlights
